#  Scoring de churn client
Le projet vise à concevoir un modèle de scoring du risque de churn pour cibler efficacement les actions de rétention client, en combinant une démarche complète de data science avec une logique d’optimisation marketing et budgétaire.

Le projet consiste à développer un modèle de prédiction du churn (résiliation client) pour l’opérateur télécom européen TelcoWave (mobile + fibre).

L’objectif est de créer un outil de scoring capable d’estimer, pour chaque client, la probabilité qu’il résilie son abonnement au cours du prochain trimestre.

Ce score permettra à l’entreprise d’anticiper les départs et d’agir en amont.
